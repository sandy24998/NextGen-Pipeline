# NextGen-Pipeline
A modern CI/CD pipeline using Jenkins, Docker, and GitHub for automated build, test, and deployment.


# Simple Flask App

This is a basic Flask application running on port 5000.

## Run locally

Install dependencies:

pip install -r requirements.txt

Run the app:

python app.py

App will start on:
http://localhost:5000

## Endpoints

/ -> main response  
/health -> health check
